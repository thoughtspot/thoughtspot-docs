= Add a Salesforce connection
:last_updated: 6/20/2020




You can add a connection to your Salesforce instance using ThoughtSpot DataFlow.

To add a new connection to Salesforce, follow these steps:

include::partial$add-database-connection.adoc[]

. After you select the Salesforce *Connection type*, the rest of the connection properties appear.
 ** xref:dataflow-salesforce-reference.adoc#dataflow-salesforce-conn-connection-name[Connection name] + Name your connection.
 ** xref:dataflow-salesforce-reference.adoc#dataflow-salesforce-conn-connection-type[Connection type] + Choose the Salesforce connection type.
 ** xref:dataflow-salesforce-reference.adoc#dataflow-salesforce-conn-user[User] + Specify the user id.
This user must have data access privileges.
 ** xref:dataflow-salesforce-reference.adoc#dataflow-salesforce-conn-password[Password] + Specify the password.
 ** xref:dataflow-salesforce-reference.adoc#dataflow-salesforce-conn-use-sandbox[Use sandbox] + Specify if connecting to copy of a production environment.

+
See xref:dataflow-salesforce-reference.adoc#connection-properties[Connection properties] for details, defaults, and examples.
. Click *Create connection*.
