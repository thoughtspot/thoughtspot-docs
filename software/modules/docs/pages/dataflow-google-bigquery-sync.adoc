 ** xref:dataflow-google-bigquery-reference.adoc#dataflow-google-bigquery-sync-column-delimiter[Column delimiter] + Specify the column delimiter character.
 ** xref:dataflow-google-bigquery-reference.adoc#dataflow-google-bigquery-sync-enclosing-character[Enclosing character] + Specify if the text columns in the source data needs to be enclosed in quotes.
 ** xref:dataflow-google-bigquery-reference.adoc#dataflow-google-bigquery-sync-escape-character[Escape character] + Specify the escape character if using a text qualifier in the source data.
 ** xref:dataflow-google-bigquery-reference.adoc#dataflow-google-bigquery-sync-fetch-size[Fetch size] + Specify the number of rows to fetch at one time, and proces in memory.

 ** xref:dataflow-google-bigquery-reference.adoc#dataflow-google-bigquery-sync-ts-load-options[TS load options] + Specify the parameters passed with the `tsload` command, in addition to the commands already included by the application.
