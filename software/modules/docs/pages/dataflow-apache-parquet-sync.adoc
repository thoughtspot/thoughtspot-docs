= Sync data through Apache Parquet connection
:last_updated: 09/14/2020




After using ThoughtSpot DataFlow to establish a connection to an Apache Parquet file system, you can create automatic data updates, to seamlessly refresh your data.

include::partial$sync-for-file-systems.adoc[]

. Specify the sync properties for files:
 ** xref:dataflow-apache-parquet-reference.adoc#dataflow-apache-parquet-sync-enclosing-character[Enclosing character] + Specify if the text columns in the source data needs to be enclosed in quotes.
 ** xref:dataflow-apache-parquet-reference.adoc#dataflow-apache-parquet-sync-[Escape character] + Specify escape characer if data uses text qualifier is mentioned.
This should be the character which escapes the text qualifier character in the source data.
 ** xref:dataflow-apache-parquet-reference.adoc#dataflow-apache-parquet-sync-[Null value] + Specifies the string literal indicates the null value for a column.
During the data load, the column value matching this string will be loaded as null in the target.
 ** xref:dataflow-apache-parquet-reference.adoc#dataflow-apache-parquet-sync-[Date style] + Specifies how to interpret the date format
 ** xref:dataflow-apache-parquet-reference.adoc#dataflow-apache-parquet-sync-[Date delimiter] + Specifies the separator used in the date format.
 ** xref:dataflow-apache-parquet-reference.adoc#dataflow-apache-parquet-sync-[Time style] + Specifies the format of the time portion in the data.
 ** xref:dataflow-apache-parquet-reference.adoc#dataflow-apache-parquet-sync-[Time delimiter] + Specifies the character used as separate the time components.
 ** xref:dataflow-apache-parquet-reference.adoc#dataflow-apache-parquet-sync-[Skip trailer rows] + Skip trailer row while processing the data
 ** xref:dataflow-apache-parquet-reference.adoc#dataflow-apache-parquet-sync-[Reader encoder] + Specify the encoding format of the source system.
 ** xref:dataflow-apache-parquet-reference.adoc#dataflow-apache-parquet-sync-[TS load options] + Specify additional parameters passed with the `tsload` command.
The format for these parameters is: + `--<param_1_name> <optional_param_1_value>`
 ** xref:dataflow-apache-parquet-reference.adoc#dataflow-apache-parquet-sync-boolean-representation[Boolean representation] + Specifies the representation of data in the boolean field

+
See xref:dataflow-apache-parquet-reference.adoc#sync-properties[Sync properties] for details, defaults, and examples.
. Save your work by clicking *Save*.
+ Alternatively, click *Save and sync now* to save your work and sync data at the same time.
