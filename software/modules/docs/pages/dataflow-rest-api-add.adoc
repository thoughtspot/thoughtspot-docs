= Add a REST API connection
:last_updated: 11/25/2020




You can add a connection to an application through REST API using ThoughtSpot DataFlow.

To add a new connection to Salesforce, follow these steps:

include::partial$add-database-connection.adoc[]

. After you select the REST API *Connection type*, the rest of the connection properties appear.
 ** xref:dataflow-rest-api-reference.adoc#dataflow-rest-api-conn-authentication-type[Authentication type] + Specify the type of authentication that is required to connect to the REST API service.
 ** xref:dataflow-rest-api-reference.adoc#dataflow-rest-api-conn-rest-api-base-url[REST API base URL] + Specify the end point URL to access REST API web-service.
 ** xref:dataflow-rest-api-reference.adoc#dataflow-rest-api-conn-user[User] + Specify the user who connects to the Rest web service.
This user must have data access privileges.
+ For BASIC authentication type only.
 ** xref:dataflow-rest-api-reference.adoc#dataflow-rest-api-conn-password[Password] + Specify the password for the User.
+ For BASIC authentication type only.
 ** xref:dataflow-rest-api-reference.adoc#dataflow-rest-api-conn-obtain-access-token[Obtain access token] + Select this option to use access key/token to connect to the REST API web-service + For OAuth 2.0 authentication type only.
 ** xref:dataflow-rest-api-reference.adoc#dataflow-rest-api-conn-access-token[Access token] + Specify the access token to authenticate REST API.
+ For OAuth 2.0 authentication type only.
 ** xref:dataflow-rest-api-reference.adoc#dataflow-rest-api-conn-refresh-token[Refresh token] + Specify the refresh token to authenticate REST API.
+ For OAuth 2.0 authentication type only.
 ** xref:dataflow-rest-api-reference.adoc#dataflow-rest-api-conn-oauth-client-id[OAuth client ID] + Specify the OAuth client ID + Displayed only when "obtain access token" check-box is selected
 ** xref:dataflow-rest-api-reference.adoc#dataflow-rest-api-conn-mask-client-secret[Mask client secret] + Specify the OAuth client secret + Displayed only when "obtain access token" check-box is selected
 ** xref:dataflow-rest-api-reference.adoc#dataflow-rest-api-conn-oauth-authorization-url[OAuth authorization URL] + Specify the OAuth authorization URL + Displayed only when "obtain access token" check-box is selected
 ** xref:dataflow-rest-api-reference.adoc#dataflow-rest-api-conn-oauth-accesstoken-url[OAuth accesstoken URL] + Specify the OAuth accesstoken URL + Displayed only when "obtain access token" check-box is selected
 ** xref:dataflow-rest-api-reference.adoc#dataflow-rest-api-conn-scope[Scope] + Specify the number of users to access the account + Displayed only when "obtain access token" check-box is selected
 ** xref:dataflow-rest-api-reference.adoc#dataflow-rest-api-conn-callback-url[Callback URL] + Secured domain URL of the repo which is used to register in REST API + Displayed only when "obtain access token" check-box is selected
 ** xref:dataflow-rest-api-reference.adoc#dataflow-rest-api-conn-rest-api-parameters[REST API parameters] + When adding REST API parameters, click *Add*, and then specify the `Parameter name`, `Value`, and if the parameter is a `Header`.

+
See xref:dataflow-rest-api-reference.adoc#connection-properties[Connection properties] for details, defaults, and examples.
. Click *Create connection*.
