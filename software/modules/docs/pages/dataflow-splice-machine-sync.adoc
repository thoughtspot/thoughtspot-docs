= Sync data through a Splice Machine connection
:last_updated: 7/7/2020

After using ThoughtSpot DataFlow to establish a connection to a Splice Machine database, you can create automatic data updates, to seamlessly refresh your data.

include::partial$sync-for-databases.adoc[]

. Specify the sync properties for Splice Machine:
 ** xref:dataflow-splice-machine-reference.adoc#dataflow-splice-machine-sync-data-extraction-mode[Data extraction mode]
 +
Specify the extraction type.
 ** xref:dataflow-splice-machine-reference.adoc#dataflow-splice-machine-sync-column-delimiter[Column delimiter]
 +
Specify the column delimiter character.
 ** xref:dataflow-splice-machine-reference.adoc#dataflow-splice-machine-sync-enclosing-character[Enclosing character]
 +
Specify if the text columns in the source data needs to be enclosed in quotes.
 ** xref:dataflow-splice-machine-reference.adoc#dataflow-splice-machine-sync-escape-character[Escape character]
 +
Specify the escape character if using a text qualifier in the source data.
 ** xref:dataflow-splice-machine-reference.adoc#dataflow-splice-machine-sync-fetch-size[Fetch size]
 +
Specify the number of rows at a time to fetch and process in memory. If you specify zero, the system extracts all rows at once.
 ** xref:dataflow-splice-machine-reference.adoc#dataflow-splice-machine-sync-ts-load-options[TS load options]
 +
Specify the parameters passed with the `tsload` command, in addition to the commands already included by the application.

+
See xref:dataflow-splice-machine-reference.adoc#sync-properties[Sync properties] for details, defaults, and examples.
. Save your work by clicking *Save*.
+ Alternatively, click *Save and sync now* to save your work and sync data at the same time.
