= Add an IBM Db2 database connection
:last_updated: 7/6/2020




You can add a connection to an IBM Db2 database using ThoughtSpot DataFlow.

Follow these steps:

include::partial$add-database-connection.adoc[]

. After you select the IBM Db2 *Connection type*, the rest of the connection properties appear.
 ** xref:dataflow-ibm-db2-reference.adoc#dataflow-ibm-db2-conn-connection-name[Connection name] + Name your connection.
 ** xref:dataflow-ibm-db2-reference.adoc#dataflow-ibm-db2-conn-connection-type[Connection type] + Choose the IBM Db2 connection type.
 ** xref:dataflow-ibm-db2-reference.adoc#dataflow-ibm-db2-conn-platform[Platform] + An environment in which a data point is executed.
Specify LUW, iSeries, or Mainframe.
 ** xref:dataflow-ibm-db2-reference.adoc#dataflow-ibm-db2-conn-host[Host] + Specify the hostname or the IP address of the Db2 system
 ** xref:dataflow-ibm-db2-reference.adoc#dataflow-ibm-db2-conn-port[Port] + Specify the port associated to the Db2 system.
 ** xref:dataflow-ibm-db2-reference.adoc#dataflow-ibm-db2-conn-user[User] + Specify the user to connect to IBM Db2.
This user must have data access privileges.
 ** xref:dataflow-ibm-db2-reference.adoc#dataflow-ibm-db2-conn-password[Password] + Specify the password.
 ** xref:dataflow-ibm-db2-reference.adoc#dataflow-ibm-db2-conn-database[Database] + Collection of information that is organized so that it can be easily accessed, managed and updated.
 ** xref:dataflow-ibm-db2-reference.adoc#dataflow-ibm-db2-conn-database-alias[Database alias] + Specify database alias for easy identification.
 ** xref:dataflow-ibm-db2-reference.adoc#dataflow-ibm-db2-sync-version[Version] + Specify the Db2 version being connected to
 ** xref:dataflow-ibm-db2-reference.adoc#dataflow-ibm-db2-sync-code-page[Code page] + A standardized code in which characters are represented for computer storage and transmission by the numbers 0 through 127.
Select either ASCII or EBCDIC.
 ** xref:dataflow-ibm-db2-reference.adoc#dataflow-ibm-db2-conn-jdbc-options[JDBC options] + Specify the options associated with the JDBC URL.

+
See xref:dataflow-ibm-db2-reference.adoc#connection-properties[Connection properties] for details, defaults, and examples.
. Click *Create connection*.
