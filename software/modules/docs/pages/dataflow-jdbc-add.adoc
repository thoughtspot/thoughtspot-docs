= Add a JDBC connection
:last_updated: 11/25/2020




With DataFlow, you can add a connection to any database when you have the appropriate JDBC driver.

To add a new connection through JDBC, follow these steps:

include::partial$add-database-connection.adoc[]

. After you select the JDBC *Connection type*, the rest of the connection properties appear.
 ** xref:dataflow-jdbc-reference.adoc#dataflow-jdbc-conn-connection-name[Connection name] + Name your connection.
 ** xref:dataflow-jdbc-reference.adoc#dataflow-jdbc-conn-connection-type[Connection type] + Choose the JDBC connection type.
 ** xref:dataflow-jdbc-reference.adoc#dataflow-jdbc-conn-driver-name[Driver name] + Specify the JDBC driver class for the database connection.
 ** xref:dataflow-jdbc-reference.adoc#dataflow-jdbc-conn-jdbc-url[JDBC URL] + Specify the jdbc url for database connection.
 ** xref:dataflow-jdbc-reference.adoc#dataflow-jdbc-conn-jdbc-driver-jar[JDBC driver jar] + Upload the jdbc jar available in local system.
 ** xref:dataflow-jdbc-reference.adoc#dataflow-jdbc-conn-jdbc-options[JDBC options] + Select if custom JDBC options to be appended to the JDBC extract command
 ** xref:dataflow-jdbc-reference.adoc#dataflow-jdbc-conn-key[Key] + Define the key parameter
 ** xref:dataflow-jdbc-reference.adoc#dataflow-jdbc-conn-value[Value] + Specify the value for key paramter
 ** xref:dataflow-jdbc-reference.adoc#dataflow-jdbc-conn-encrypt[Encrypt] + Enrypt the details for secured authentication + Select the check-box to enrypt the details
 ** xref:dataflow-jdbc-reference.adoc#dataflow-jdbc-conn-schema-object-name-enclosing-character[Schema object name enclosing character] + Specify the schema object name enclosing character.

+
See xref:dataflow-jdbc-reference.adoc#connection-properties[Connection properties] for details, defaults, and examples.
. Click *Create connection*.
