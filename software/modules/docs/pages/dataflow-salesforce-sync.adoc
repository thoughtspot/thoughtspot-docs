= Sync data through a Salesforce connection
:last_updated: 7/13/2020




After using ThoughtSpot DataFlow to establish a connection to  Salesforce, you can create automatic data updates, to seamlessly refresh your data.

include::partial$sync-for-salesforce.adoc[]

. Specify the sync properties for Salesforce:
 ** xref:dataflow-salesforce-reference.adoc#dataflow-salesforce-sync-column-delimiter[Column delimiter] + Specify the column delimiter character.
 ** xref:dataflow-salesforce-reference.adoc#dataflow-salesforce-sync-enclosing-character[Enclosing character] + Specify if the text columns in the source data must be enclosed in quotes.
 ** xref:dataflow-salesforce-reference.adoc#dataflow-salesforce-sync-escape-character[Escape character] + Specify the escape character if using a text qualifier in the source data.
 ** xref:dataflow-salesforce-reference.adoc#dataflow-salesforce-sync-fetch-size[Fetch size] + Specify the number of rows fetched into memory at the same time.
If the value is 0, system fetches all rows at the same time.
 ** xref:dataflow-salesforce-reference.adoc#dataflow-salesforce-sync-ts-load-options[TS load options] + Specify the parameters passed with the `tsload` command, in addition to the commands already included by the application.

+
See xref:dataflow-salesforce-reference.adoc#sync-properties[Sync properties] for details, defaults, and examples.
. Save your work by clicking *Save*.
+ Alternatively, click *Save and sync now* to save your work and sync data at the same time.
