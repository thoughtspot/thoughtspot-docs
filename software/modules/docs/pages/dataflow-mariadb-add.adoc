= Add a MariaDB database connection
:last_updated: 7/6/2020




You can add a connection to a MariaDB database using ThoughtSpot DataFlow.

Follow these steps:

include::partial$add-database-connection.adoc[]

. After you select the MariaDB *Connection type*, the rest of the connection properties appear.

* xref:dataflow-mariadb-reference.adoc#dataflow-mariadb-conn-connection-name[Connection name] + Name your connection.
 ** xref:dataflow-mariadb-reference.adoc#dataflow-mariadb-conn-connection-type[Connection type] + Choose the MariaDB connection type.
 ** xref:dataflow-mariadb-reference.adoc#dataflow-mariadb-conn-host[Host] + Specify the hostname or the IP address of the MariaDB system
 ** xref:dataflow-mariadb-reference.adoc#dataflow-mariadb-conn-port[Port] + Specify the port associated to the MariaDB system
 ** xref:dataflow-mariadb-reference.adoc#dataflow-mariadb-conn-user[User] + Specify the user id that will be used to connect to the MariaDB system.
This user should have necessary privileges to access the data in the databases.
 ** xref:dataflow-mariadb-reference.adoc#dataflow-mariadb-conn-password[Password] + Specify the password for the User
 ** xref:dataflow-mariadb-reference.adoc#dataflow-mariadb-conn-database-version[Database version] + Specify the MariaDB version being connected to.
 ** xref:dataflow-mariadb-reference.adoc#dataflow-mariadb-conn-jdbc-options[JDBC options] + Specify the options associated with the JDBC URL.

See xref:dataflow-mariadb-reference.adoc#connection-properties[Connection properties] for details, defaults, and examples.

. Click *Create connection*.
