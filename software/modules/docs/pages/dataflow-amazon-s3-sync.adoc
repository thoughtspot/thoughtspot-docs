= Sync data through Amazon S3 connection
:last_updated: 7/7/2020




After using ThoughtSpot DataFlow to establish a connection to an Amazon S3 file system, you can create automatic data updates, to seamlessly refresh your data.

include::partial$sync-for-file-systems.adoc[]

. Specify the sync properties for files:
** xref:dataflow-amazon-s3-reference.adoc#dataflow-amazon-s3-sync-file-name[File name] + Specify the name of the file.
** xref:dataflow-amazon-s3-reference.html#dataflow-amazon-s3-sync-enable-archive-on-success[Enable archive on success] + Specify if must archive file after operation succeeds.
 ** xref:dataflow-amazon-s3-reference.adoc#dataflow-amazon-s3-sync-delete-file-on-success[Delete file on success] + Specify if must delete file after operation succeeds.
 ** xref:dataflow-amazon-s3-reference.adoc#dataflow-amazon-s3-sync-column-delimiter[Column delimiter] + Specify the column delimiter character.
 ** xref:dataflow-amazon-s3-reference.adoc#dataflow-amazon-s3-sync-skip-header-rows[Skip header rows] + Skip the specified number of header rows when loading data.
 ** xref:dataflow-amazon-s3-reference.adoc#dataflow-amazon-s3-sync-compression[Compression] + Specify if the file has compression, and type of compresssion.
 ** xref:dataflow-amazon-s3-reference.adoc#dataflow-amazon-s3-sync-row-delimiter[Row delimiter] + Specify the end of the row character in the extracted data.
 ** xref:dataflow-amazon-s3-reference.adoc#dataflow-amazon-s3-sync-enclosing-character[Enclosing character] + Specify if text columns in the source data use enclosing quotation marks, and their type.
 ** xref:dataflow-amazon-s3-reference.adoc#dataflow-amazon-s3-sync-escape-character[Escape character] + Specify the escape character if using a text qualifier in the source data.
 ** xref:dataflow-amazon-s3-reference.adoc#dataflow-amazon-s3-sync-null-value[Null value] + Specifies the string literal indicates the null value for a column.
During the data load, the column value matching this string will be loaded as null in the target.
 ** xref:dataflow-amazon-s3-reference.adoc#dataflow-amazon-s3-sync-date-style[Date style] + Specifies how to interpret the date format.
 ** xref:dataflow-amazon-s3-reference.adoc#dataflow-amazon-s3-sync-date-delimiter[Date delimiter] + Specifies the separator used in the date format.
 ** xref:dataflow-amazon-s3-reference.adoc#dataflow-amazon-s3-sync-time-style[Time style] + Specifies the format of the time portion in the data.
 ** xref:dataflow-amazon-s3-reference.adoc#dataflow-amazon-s3-sync-time-delimiter[Time delimiter] + Specifies the character used as separate the time components.
 ** xref:dataflow-amazon-s3-reference.adoc#dataflow-amazon-s3-sync-skip-trailer-rows[Skip trailer rows] + Skip the number of trailer rows specified while loading the data.
 ** xref:dataflow-amazon-s3-reference.adoc#dataflow-amazon-s3-sync-ts-load-options[TS load options] + Specify the parameters passed with the `tsload` command, in addition to the commands already included by the application.
 ** xref:dataflow-amazon-s3-reference.adoc#dataflow-amazon-s3-sync-boolean-representation[Boolean representation] + Specifies the representation of data in the boolean field.

+
See xref:dataflow-amazon-s3-reference.adoc#sync-properties[Sync properties] for details, defaults, and examples.
. Save your work by clicking *Save*.
+ Alternatively, click *Save and sync now* to save your work and sync data at the same time.
