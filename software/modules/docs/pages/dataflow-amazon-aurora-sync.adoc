= Sync data through an Amazon Aurora connection
:last_updated: 7/03/2020

After using ThoughtSpot DataFlow to establish a connection to an Amazon Aurora database, you can create automatic data updates, to seamlessly refresh your data.

include::partial$sync-for-databases.adoc[]

. Specify the sync properties for Amazon Aurora:
 ** xref:dataflow-amazon-aurora-reference.adoc#dataflow-amazon-aurora-sync-column-delimiter[Column delimiter] + Specify the column delimiter character.
 ** xref:dataflow-amazon-aurora-reference.adoc#dataflow-amazon-aurora-sync-enclosing-character[Enclosing character] + Specify if the text columns in the source data needs to be enclosed in quotes.
 ** xref:dataflow-amazon-aurora-reference.adoc#dataflow-amazon-aurora-sync-escape-character[Escape character] + Specify the escape character if using a text qualifier in the source data.
 ** xref:dataflow-amazon-aurora-reference.adoc#dataflow-amazon-aurora-sync-fetch-size[Fetch size] + Specify the number of rows to be fetched at a time and processed in memory.
If the value specified is zero then, all rows are extracted at once.
 ** xref:dataflow-amazon-aurora-reference.adoc#dataflow-amazon-aurora-sync-ts-load-options[TS load options] + Specify the parameters passed with the `tsload` command, in addition to the commands already included by the application.

+
See xref:dataflow-amazon-aurora-reference.adoc#sync-properties[Sync properties] for details, defaults, and examples.
. Save your work by clicking *Save*.
+ Alternatively, click *Save and sync now* to save your work and sync data at the same time.
