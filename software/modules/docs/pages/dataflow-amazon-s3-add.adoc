= Add an Amazon S3 connection
:last_updated: 02/19/2021
:linkattrs:
:experimental:

You can add a connection to the Amazon S3 file system using ThoughtSpot DataFlow.

Follow these steps:

include::partial$add-database-connection.adoc[]

. After you select the *Connection type*, the rest of the connection properties appear.
 ** xref:dataflow-amazon-s3-reference.adoc#dataflow-amazon-s3-conn-connection-name[Connection name] + Name your connection.
 ** xref:dataflow-amazon-s3-reference.adoc#dataflow-amazon-s3-conn-connection-type[Connection type] + Choose the Amazon S3 connection type.
 ** xref:dataflow-amazon-s3-reference.adoc#dataflow-amazon-s3-conn-amazon-s3-url[Amazon S3 URL] + Specify the Amazon S3 hostname link.
 ** xref:dataflow-amazon-s3-reference.adoc#dataflow-amazon-s3-conn-bucket[Bucket] + Specify the bucket.
+ An Amazon S3 bucket name is globally unique, and the namespace is shared by all AWS accounts.
 ** xref:dataflow-amazon-s3-reference.adoc#dataflow-amazon-s3-conn-access-key[Access key] + Specify the access key ID generated when creating AWS security credentials.
 ** xref:dataflow-amazon-s3-reference.adoc#dataflow-amazon-s3-conn-secret-key[Secret key] + Specify the secrete access key generated when creating AWS security credentials.

+
See xref:dataflow-amazon-s3-reference.adoc#connection-properties[Connection properties] for details, defaults, and examples.
. Click *Create connection*.
