 ** xref:dataflow-google-cloud-storage-reference.adoc#dataflow-google-cloud-storage-conn-connection-name[Connection name] + Name your connection.
 ** xref:dataflow-google-cloud-storage-reference.adoc#dataflow-google-cloud-storage-conn-connection-type[Connection type] + Choose the Google Cloud storage connection type.
 ** xref:dataflow-google-cloud-storage-reference.adoc#dataflow-google-cloud-storage-conn-project-id[Project ID] + The identification number given to particular project and its always unique.
 ** xref:dataflow-google-cloud-storage-reference.adoc#dataflow-google-cloud-storage-conn-authentication-type-[Authentication type] + It can be either Service Account or Access Tokens + Mandatory field.
 ** xref:dataflow-google-cloud-storage-reference.adoc#dataflow-google-cloud-storage-conn-service-account-key-or-access-token[Service account key or access token] + Provide the Service Account key when authentication type is selected as Service account and token when access token is selected as authentication type.
 ** xref:dataflow-google-cloud-storage-reference.adoc#dataflow-google-cloud-storage-conn-bucket-[Bucket] + To specify GCS bucket name + Mandatory field.
 ** xref:dataflow-google-cloud-storage-reference.adoc#dataflow-google-cloud-storage-conn-file-location-[File location] + Specify directory of source on agent location + Mandatory field.
 ** xref:dataflow-google-cloud-storage-reference.adoc#dataflow-google-cloud-storage-conn-use-proxy[Use proxy] + If required, to use a proxy, select the check box Use Proxy and provide the details + Optional field.
 ** xref:dataflow-google-cloud-storage-reference.adoc#dataflow-google-cloud-storage-conn-host[Host] + Specify the hostname or the IP address of the GCS system + Mandatory field.
When using proxy authentication only.
 ** xref:dataflow-google-cloud-storage-reference.adoc#dataflow-google-cloud-storage-conn-port[Port] + Specify the port associated to the GCS system + Mandatory field.
When using proxy authentication only.
 ** xref:dataflow-google-cloud-storage-reference.adoc#dataflow-google-cloud-storage-conn-protocol[Protocol] + It can be either http or https + Mandatory field.
When using proxy authentication only.

+
See xref:dataflow-google-cloud-storage-reference.adoc#connection-properties[Connection properties] for details, defaults, and examples.

. Click *Create connection*.
