= Add an Amazon Aurora database connection
:last_updated: 7/3/2020

You can add a connection to an Amazon Aurora database using ThoughtSpot DataFlow.

Follow these steps:

include::partial$add-database-connection.adoc[]

. After you select the Amazon Aurora *Connection type*, the rest of the connection properties appear.
 ** xref:dataflow-amazon-aurora-reference.adoc#dataflow-amazon-aurora-conn-connection-name[Connection name]
 +
Name your connection.
 ** xref:dataflow-amazon-aurora-reference.adoc#dataflow-amazon-aurora-conn-connection-type[Connection type]
 +
Choose the Amazon Aurora connection type.
 ** xref:dataflow-amazon-aurora-reference.adoc#dataflow-amazon-aurora-conn-host[Host]
 +
Specify the hostname or IP address of the Aurora system.
 ** xref:dataflow-amazon-aurora-reference.adoc#dataflow-amazon-aurora-conn-port[Port]
 +
Specify the port associated with Amazon Aurora connections.
 ** xref:dataflow-amazon-aurora-reference.adoc#dataflow-amazon-aurora-conn-user[User]
 +
Specify the user to connect to Amazon Aurora.
This user must have data access privileges.
 ** xref:dataflow-amazon-aurora-reference.adoc#dataflow-amazon-aurora-conn-password[Password]
 +
Specify the password for the User.
 ** xref:dataflow-amazon-aurora-reference.adoc#dataflow-amazon-aurora-conn-jdbc-options[JDBC options]
 +
Specify the options associated with the JDBC URL.

+
See xref:dataflow-amazon-aurora-reference.adoc#connection-properties[Connection properties] for details, defaults, and examples.
. Click *Create connection*.
