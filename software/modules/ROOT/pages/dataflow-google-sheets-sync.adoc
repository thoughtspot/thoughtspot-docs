. Specify the sync properties for Google Sheets:
+
See xref:dataflow-google-sheets-reference.adoc#sync-properties[Sync properties] for details, defaults, and examples.
+
xref:dataflow-google-sheets-reference.adoc#dataflow-google-sheets-sync-column-delimiter[Column delimiter]:: Specify the column delimiter character.

xref:dataflow-google-sheets-reference.adoc#dataflow-google-sheets-sync-enclosing-character[Enclosing character]:: Specify if the text columns in the source data must be in quotes, and the quote types.

xref:dataflow-google-sheets-reference.adoc#dataflow-google-sheets-sync-escape-character[Escape character]:: Specify the escape character if using a text qualifier in the source data.

xref:dataflow-google-sheets-reference.adoc#dataflow-google-sheets-sync-fetch-size[Fetch size]:: Specify the number of rows to fetch at one time, and process in memory.
+
If the value specified is `0`, then DataFlow extracts all rows at the same time.

xref:dataflow-google-sheets-reference.adoc#dataflow-google-sheets-sync-max-ignored-rows[Max ignored rows]:: Terminate the transaction when encounter `n` ignored rows.

xref:dataflow-google-sheets-reference.adoc#dataflow-google-sheets-sync-ts-load-options[TS load options]:: Specifies the parameters passed with the `tsload` command, in addition to the commands already included by the application.
+
The format for these parameters is:
+
----
--<param_1_name><optonal_param_1_value>
--<param_2_name><optonal_param_2_value>
----


. Save your work by clicking *Save*.
+
Alternatively, click *Save and sync now* to save your work and sync data at the same time.
