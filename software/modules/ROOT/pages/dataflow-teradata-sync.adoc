= Sync data through a Teradata connection
:last_updated: 9/9/2021
:experimental:
:linkattrs:
:page-aliases: /data-integrate/dataflow/dataflow-teradata-sync.adoc

include::partial$liveboards-announcement.adoc[]


After using ThoughtSpot DataFlow to establish a connection to a Teradata database, you can create automatic data updates, to seamlessly refresh your data.

include::partial$sync-for-databases.adoc[]

. Specify the sync properties for Teradata:
+
See xref:dataflow-teradata-reference.adoc#sync-properties[Sync properties] for details, defaults, and examples.
+
// ![Enter connection details]({{ site.baseurl }}/images/dataflow-teradata-sync.png "Enter connection details")
xref:dataflow-teradata-reference.adoc#dataflow-teradata-sync-data-extraction-mode[Data extraction mode]:: Specify the extraction type.
xref:dataflow-teradata-reference.adoc#dataflow-teradata-sync-column-delimiter[Column delimiter]:: Specify the column delimiter character.
xref:dataflow-teradata-reference.adoc#dataflow-teradata-sync-null-value[Null value]:: Specifies the string literal that indicates the null value in the extracted data.
During the data load, the column value matching this string loads as null in the target.
xref:dataflow-teradata-reference.adoc#dataflow-teradata-sync-enclosing-character[Enclosing character]:: Specify if the text columns in the source data needs to be enclosed in quotes.
xref:dataflow-teradata-reference.adoc#dataflow-teradata-sync-escape-character[Escape character]:: Specify the escape character if using a text qualifier in the source data.
xref:dataflow-teradata-reference.adoc#dataflow-teradata-sync-fetch-size[Fetch size]:: Specify the number of rows at a time to fetch and process in memory.
If you specify zero, the system extracts all rows at once.
xref:dataflow-teradata-reference.adoc#dataflow-teradata-sync-max-ignored-rows[Max ignored rows]:: Abort the transaction after encountering 'n' ignored rows.
xref:dataflow-teradata-reference.adoc#dataflow-teradata-sync-ts-load-options[TS load options]:: Specify the parameters passed with the `tsload` command, in addition to the commands already included by the application.

. Save your work by clicking *Save*.
+
Alternatively, click *Save and sync now* to save your work and sync data at the same time.
