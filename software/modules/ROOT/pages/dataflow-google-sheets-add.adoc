= Add a Google Sheets database connection
:last_updated: 09/09/2021
:linkattrs:
:experimental:
:page-aliases: /data-integrate/dataflow/dataflow-google-sheets-add.adoc

You can add a connection to Google Sheets and use it as a database in ThoughtSpot DataFlow.

Follow these steps:

include::partial$add-database-connection.adoc[]

. After you select the Google Sheets *Connection type*, the rest of the connection properties appear.
+
See xref:dataflow-google-sheets-reference.adoc#connection-properties[Connection properties] for details, defaults, and examples.
+
xref:dataflow-google-sheets-reference.adoc#dataflow-google-sheets-conn-connection-name[Connection name]:: Name your connection.

xref:dataflow-google-sheets-reference.adoc#dataflow-google-sheets-conn-connection-type[Connection type]:: Choose the Google Sheets connection type.

xref:dataflow-google-sheets-reference.adoc#dataflow-google-sheets-conn-access-token[Access token]:: Specify the access token to authenticate Google Sheets.

xref:dataflow-google-sheets-reference.adoc#dataflow-google-sheets-conn-refresh-token[Refresh token]:: Specify the refresh token to authenticate Google Sheets.

xref:dataflow-google-sheets-reference.adoc#dataflow-google-sheets-conn-obtain-access-token[Obtain access token]:: Select this option to use access key/token to connect to the Google Sheets web-service

xref:dataflow-google-sheets-reference.adoc#dataflow-google-sheets-conn-oauth-client-id[OAuth client ID]:: Specify the OAuth client ID

xref:dataflow-google-sheets-reference.adoc#dataflow-google-sheets-conn-oauth-client-secret[OAuth client secret]:: Specify the OAuth client secret

xref:dataflow-google-sheets-reference.adoc#dataflow-google-sheets-conn-callback-url[Callback URL]:: Secured domain URL of the repo used to register in Google Sheets

xref:dataflow-google-sheets-reference.adoc#dataflow-google-sheets-conn-scope[Scope]:: Specify the number of users to access the account


. Click *Create connection*.
