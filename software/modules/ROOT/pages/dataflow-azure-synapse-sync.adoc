= Sync data through Azure Synapse DataFlow connection
:last_updated: 6/17/2020
:experimental:
:linkattrs:
:page-aliases: /data-integrate/dataflow/dataflow-azure-synapse-sync.adoc



After using ThoughtSpot DataFlow to establish a connection to an Azure Synapse database, you can create automatic data updates, to seamlessly refresh your data.

include::partial$sync-for-databases.adoc[]

. Specify the sync properties for Azure Synapse:
+
See xref:dataflow-azure-synapse-reference.adoc#sync-properties[Sync properties] for details, defaults, and examples.
+
// image:dataflow-azure-synapse-sync.png[Enter sync details]
xref:dataflow-azure-synapse-reference.adoc#dataflow-azure-synapse-sync-data-extraction-mode[Data extraction mode]:: Specify the extraction type.
xref:dataflow-azure-synapse-reference.adoc#dataflow-azure-synapse-sync-column-delimiter[Column delimiter]:: Specify the column delimiter character.
xref:dataflow-azure-synapse-reference.adoc#dataflow-azure-synapse-sync-null-value[Null value]:: Specify the string literal that represents NULL on the source.
When loading data, the system replaces this with NULL.
Available only when *Data extraction mode* is _BCP_.
xref:dataflow-azure-synapse-reference.adoc#dataflow-azure-synapse-sync-enclosing-character[Enclosing character]:: Specify if text columns in the source data are enclosed in quotes;
if yes, single quotes or double quotes.
xref:dataflow-azure-synapse-reference.adoc#dataflow-azure-synapse-sync-escape-character[Escape character]:: Specify the escape character if using a text qualifier in the source data.
xref:dataflow-azure-synapse-reference.adoc#dataflow-azure-synapse-sync-fetch-size[Fetch size]:: Specify the number of rows fetched into memory at the same time.
If the value is 0, system fetches all rows at the same time.
Available only when *Data extraction mode* is _JDBC_.
xref:dataflow-azure-synapse-reference.adoc#dataflow-azure-synapse-sync-ts-load-options[TS load options]:: Specify the parameters passed with the `tsload` command, in addition to the commands already included by the application.

. Save your work by clicking *Save*.
+
Alternatively, click *Save and sync now* to save your work and sync data at the same time.
