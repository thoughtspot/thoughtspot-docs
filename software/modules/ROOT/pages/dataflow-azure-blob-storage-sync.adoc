= Sync data through Azure Blob Storage connection
:last_updated: 7/7/2020
:experimental:
:linkattrs:
:page-aliases: /data-integrate/dataflow/dataflow-azure-blob-storage-sync.adoc


After using ThoughtSpot DataFlow to establish a connection to an Azure Blob Storage file system, you can create automatic data updates, to seamlessly refresh your data.

Starting with ThoughtSpot Release 6.3, you can use the default _Blob Storage_, or _Delta Lake GEN2 Storage_ [.label.label-beta]#New# options.

include::partial$sync-for-file-systems.adoc[]

. Specify the sync properties for Azure Blob Storage:
+
See xref:dataflow-azure-blob-storage-reference.adoc#sync-properties[Sync properties] for details, defaults, and examples.
+
xref:dataflow-azure-blob-storage-reference.adoc#dataflow-azure-blob-storage-sync-file-name[File name]:: Specify the name of the file.
+
Mandatory field.
xref:dataflow-azure-blob-storage-reference.adoc#dataflow-azure-blob-storage-sync-column-delimiter[Column delimiter]::
 Specifies the delimiter between the columns in the source file.
+
Mandatory field.
xref:dataflow-azure-blob-storage-reference.adoc#dataflow-azure-blob-storage-sync-enable-archive-on-success[Enable archive on success]:: Specify if file needs to be archived once it is succeeded.
+
Optional field.
xref:dataflow-azure-blob-storage-reference.adoc#dataflow-azure-blob-storage-sync-delete-on-success[Delete on success]:: Specify if file needs to be deleted after execution is successful.
+
Optional field.
xref:dataflow-azure-blob-storage-reference.adoc#dataflow-azure-blob-storage-sync-compression[Compression]:: Specify this if the file is compressed and what kind of compressed file it is.
+
Mandatory field.
xref:dataflow-azure-blob-storage-reference.adoc#dataflow-azure-blob-storage-sync-enclosing-character[Enclosing character]:: Specify if the text columns in the source data is enclosed in quotes.
xref:dataflow-azure-blob-storage-reference.adoc#dataflow-azure-blob-storage-sync-escape-character[Escape character]::
 Specify the escape character if using a text qualifier in the source data.
xref:dataflow-azure-blob-storage-reference.adoc#dataflow-azure-blob-storage-sync-null-value[Null value]:: Specifies the string literal indicates the null value for a column.
During the data load, the column value matching this string will be loaded as null in the target.
xref:dataflow-azure-blob-storage-reference.adoc#dataflow-azure-blob-storage-sync-date-style[Date style]:: Specifies how to interpret the date format.
+
Optional field.
xref:dataflow-azure-blob-storage-reference.adoc#dataflow-azure-blob-storage-sync-date-delimiter[Date delimiter]::
 Specifies the separator used in the date format.
xref:dataflow-azure-blob-storage-reference.adoc#dataflow-azure-blob-storage-sync-time-style[Time style]::
 Specifies the format of the time portion in the data.
xref:dataflow-azure-blob-storage-reference.adoc#dataflow-azure-blob-storage-sync-time-delimiter[Time delimiter]::
 Specifies the character used as separate the time components.
xref:dataflow-azure-blob-storage-reference.adoc#dataflow-azure-blob-storage-sync-skip-trailer-rows[Skip trailer rows]::
Skip the number of trailer rows specified while loading the data.
+
Optional field.
xref:dataflow-azure-blob-storage-reference.adoc#dataflow-azure-blob-storage-sync-ts-load-options[TS load options]::
 Specify the parameters passed with the `tsload` command, in addition to the commands already included by the application.
xref:dataflow-azure-blob-storage-reference.adoc#dataflow-azure-blob-storage-sync-boolean-representation[Boolean representation]::
 Specifies the representation of data in the boolean field.

. Save your work by clicking *Save*.
+
Alternatively, click *Save and sync now* to save your work and sync data at the same time.
