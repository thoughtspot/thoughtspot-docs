= Add a Dremio connection
:last_updated: 9/9/2021
:linkattrs:
:experimental:
:page-aliases: /data-integrate/dataflow/dataflow-dremeo-add.adoc

You can add a connection to the Dremio database using ThoughtSpot DataFlow.

Follow these steps:

include::partial$add-database-connection.adoc[]

. After you select the Dremio *Connection type*, the rest of the connection properties appear.
+
See xref:dataflow-dremio-reference.adoc#connection-properties[Connection properties] for details, defaults, and examples.
+
xref:dataflow-dremio-reference.adoc#dataflow-dremio-conn-connection-name[Connection name]:: Name your connection.

xref:dataflow-dremio-reference.adoc#dataflow-dremio-conn-connection-type[Connection type]:: Choose the Dremio connection type.

xref:dataflow-dremio-reference.adoc#dataflow-dremio-conn-host[Host]:: Specify the hostname or the IP address of the Dremio system

xref:dataflow-dremio-reference.adoc#dataflow-dremio-conn-port[Port]:: Specify the port associated to the Dremio system

xref:dataflow-dremio-reference.adoc#dataflow-dremio-conn-user[User]:: Specify the user to connect to Dremio. This user must have data access privileges.

xref:dataflow-dremio-reference.adoc#dataflow-dremio-conn-password[Password]:: Specify the password for the User.

xref:dataflow-dremio-reference.adoc#dataflow-dremio-conn-jdbc-options[JDBC options]:: Specify the options associated with the JDBC URL.

. Click *Create connection*.
