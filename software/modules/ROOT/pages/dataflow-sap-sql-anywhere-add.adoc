= Add an SAP SQL Anywhere database connection
:last_updated: 09/09/2021
:linkattrs:
:experimental:
:page-aliases: /data-integrate/dataflow/dataflow-sap-sql-anywhere-add.adoc

You can add a connection to an SAP SQL Anywhere database using ThoughtSpot DataFlow.

Follow these steps:

include::partial$add-database-connection.adoc[]

. After you select the SAP SQL Anywhere *Connection type*, the rest of the connection properties appear.
+
See xref:dataflow-sap-sql-anywhere-reference.adoc#connection-properties[Connection properties] for details, defaults, and examples.
+
xref:dataflow-sap-sql-anywhere-reference.adoc#dataflow-sap-sql-anywhere-conn-connection-name[Connection name]:: Name your connection.
xref:dataflow-sap-sql-anywhere-reference.adoc#dataflow-sap-sql-anywhere-conn-connection-type[Connection type]:: Choose the SAP SQL Anywhere connection type.
xref:dataflow-sap-sql-anywhere-reference.adoc#dataflow-sap-sql-anywhere-conn-host[Host]:: Specify the hostname or the IP address of the Sybase system.
xref:dataflow-sap-sql-anywhere-reference.adoc#dataflow-sap-sql-anywhere-conn-port[Port]:: Specify the port associated to the Sybase system.
xref:dataflow-sap-sql-anywhere-reference.adoc#dataflow-sap-sql-anywhere-conn-user[User]:: Specify the user id that will be used to connect to the Sybase system.
This user should have necessary privileges to access the data in the databases.
xref:dataflow-sap-sql-anywhere-reference.adoc#dataflow-sap-sql-anywhere-conn-password[Password]:: Specify the password for the User.
+
////
xref:dataflow-sap-sql-anywhere-reference.adoc#dataflow-sap-sql-anywhere-conn-version[Version]:: Specify the version of Sybase.
////
xref:dataflow-sap-sql-anywhere-reference.adoc#dataflow-sap-sql-anywhere-conn-jdbc-options[JDBC options]:: Specify the options associated with the JDBC URL.

. Click *Create connection*.

'''
