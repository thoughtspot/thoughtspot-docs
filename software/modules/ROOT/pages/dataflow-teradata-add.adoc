= Add a Teradata DataFlow connection
:last_updated: 7/7/2020

You can add a connection to a Teradata database using ThoughtSpot DataFlow.

Follow these steps:

include::partial$add-database-connection.adoc[]

. After you select the Teradata *Connection type*, the rest of the connection properties appear.
 ** xref:dataflow-teradata-reference.adoc#dataflow-teradata-conn-connection-name[Connection name]
 +
Name your connection.
 ** xref:dataflow-teradata-reference.adoc#dataflow-teradata-conn-connection-type[Connection type]
 +
Choose the Teradata connection type.
 ** xref:dataflow-teradata-reference.adoc#dataflow-teradata-conn-host[Host]
 +
Specify the hostname or the IP address of the Teradata system.
 ** xref:dataflow-teradata-reference.adoc#dataflow-teradata-conn-port[Port]
 +
Specify the port associated to the Teradata system.
 ** xref:dataflow-teradata-reference.adoc#dataflow-teradata-conn-user[User]
 +
Specify the user id that will be used to connect to the Teradata system. This user should have necessary privileges to access the data in the databases.
 ** xref:dataflow-teradata-reference.adoc#dataflow-teradata-conn-password[Password]
 +
Specify the password for the User.
 ** xref:dataflow-teradata-reference.adoc#dataflow-teradata-conn-version[Version]
 +
Specify the version of Teradata.
 ** xref:dataflow-teradata-reference.adoc#dataflow-teradata-conn-jdbc-options[JDBC options]
 +
Specify the options associated with the JDBC URL.

+
See xref:dataflow-teradata-reference.adoc#connection-properties[Connection properties] for details, defaults, and examples.
. Click *Create connection*.
