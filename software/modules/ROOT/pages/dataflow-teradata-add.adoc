= Add a Teradata database connection
:last_updated: 03/10/2022
:linkattrs:
:experimental:
:page-aliases: /data-integrate/dataflow/dataflow-teradata-add.adoc
:description: You can add a connection to a Teradata database using ThoughtSpot DataFlow.

You can add a connection to a Teradata database using ThoughtSpot DataFlow.

Follow these steps:

include::partial$add-database-connection.adoc[]

. After you select the Teradata *Connection type*, the rest of the connection properties appear.
+
See xref:dataflow-teradata-reference.adoc#connection-properties[Connection properties] for details, defaults, and examples.
+
xref:dataflow-teradata-reference.adoc#dataflow-teradata-conn-connection-name[Connection name]:: Name your connection.
xref:dataflow-teradata-reference.adoc#dataflow-teradata-conn-connection-type[Connection type]:: Choose the Teradata connection type.
xref:dataflow-teradata-reference.adoc#dataflow-teradata-conn-host[Host]:: Specify the hostname or the IP address of the Teradata system.
xref:dataflow-teradata-reference.adoc#dataflow-teradata-conn-port[Port]:: Specify the port associated to the Teradata system.
xref:dataflow-teradata-reference.adoc#dataflow-teradata-conn-authentication-type[Authentication type]:: Specifies the type of security protocol to connect to the instance.
xref:dataflow-teradata-reference.adoc#dataflow-teradata-conn-user[User]:: Specify the user id that will be used to connect to the Teradata system.
This user should have necessary privileges to access the data in the databases.
xref:dataflow-teradata-reference.adoc#dataflow-teradata-conn-password[Password]:: Specify the password for the User.
xref:dataflow-teradata-reference.adoc#dataflow-teradata-conn-database[Database]:: Collection of information that is organized so that it can be easily accessed, managed and updated.
xref:dataflow-teradata-reference.adoc#dataflow-teradata-conn-jdbc-options[JDBC options]:: Specify the options associated with the JDBC URL.
xref:dataflow-teradata-reference.adoc#dataflow-teradata-conn-version[Version]:: Specify the version of Teradata.

. Select *Create connection*.
