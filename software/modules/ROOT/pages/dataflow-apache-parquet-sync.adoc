= Sync data through Apache Parquet connection
:last_updated: 09/10/2021
:experimental:
:linkattrs:
:page-aliases: /data-integrate/dataflow/dataflow-apache-parquet-sync.adoc


After using ThoughtSpot DataFlow to establish a connection to an Apache Parquet file system, you can create automatic data updates, to seamlessly refresh your data.

include::partial$sync-for-file-systems.adoc[]

. Specify the sync properties for Apache Parquet:
+
See xref:dataflow-apache-parquet-reference.adoc#sync-properties[Sync properties] for details, defaults, and examples.
+
////
xref:dataflow-apache-parquet-reference.adoc#dataflow-apache-parquet-sync-enclosing-character[Enclosing character]:: Specify if the text columns in the source data needs to be enclosed in quotes.
xref:dataflow-apache-parquet-reference.adoc#dataflow-apache-parquet-sync-escape-character[Escape character]:: Specify escape character if data uses text qualifier is mentioned.
This should be the character which escapes the text qualifier character in the source data.
////
xref:dataflow-apache-parquet-reference.adoc#dataflow-apache-parquet-sync-null-value[Null value]:: Specifies the string literal indicates the null value for a column.
During the data load, the column value matching this string will be loaded as null in the target.
xref:dataflow-apache-parquet-reference.adoc#dataflow-apache-parquet-sync-date-style[Date style]:: Specifies how to interpret the date format
xref:dataflow-apache-parquet-reference.adoc#dataflow-apache-parquet-sync-date-delimiter[Date delimiter]:: Specifies the separator used in the date format.
xref:dataflow-apache-parquet-reference.adoc#dataflow-apache-parquet-sync-time-style[Time style]:: Specifies the format of the time portion in the data.
xref:dataflow-apache-parquet-reference.adoc#dataflow-apache-parquet-sync-time-delimiter[Time delimiter]:: Specifies the character used as separate the time components.
xref:dataflow-apache-parquet-reference.adoc#dataflow-apache-parquet-sync-boolean-representation[Boolean representation]:: Specifies the representation of data in the boolean field.
xref:dataflow-apache-parquet-reference.adoc#dataflow-apache-parquet-sync-skip-trailer-rows[Skip trailer rows]:: Skip trailer row while processing the data
xref:dataflow-apache-parquet-reference.adoc#dataflow-apache-parquet-sync-reader-encoder[Reader encoder]:: Specify the encoding format of the source system.
xref:dataflow-apache-parquet-reference.adoc#dataflow-apache-parquet-sync-parallel-thread-limit[Parallel thread limit]:: Specify the number of parallel threads to unload data.

////
xref:dataflow-apache-parquet-reference.adoc#dataflow-apache-parquet-sync-tsload-options[TS load options]:: Specify additional parameters passed with the `tsload` command.
+
The format for these parameters is:
+
[source]
----
--<param_1_name> <optional_param_1_value>
--<param_2_name> <optional_param_2_value>
----
////




. Save your work by clicking *Save*.
+
Alternatively, click *Save and sync now* to save your work and sync data at the same time.
