= Add an SAP HANA database connection
:last_updated: 7/7/2020
:experimental:
:linkattrs:
:page-aliases: /data-integrate/dataflow/dataflow-sap-hana-add.adoc



You can add a connection to an SAP HANA database using ThoughtSpot DataFlow.

Follow these steps:

include::partial$add-database-connection.adoc[]

. After you select the SAP HANA *Connection type*, the rest of the connection properties appear.
+
See xref:dataflow-sap-hana-reference.adoc#connection-properties[Connection properties] for details, defaults, and examples.
+
xref:dataflow-sap-hana-reference.adoc#dataflow-sap-hana-conn-connection-name[Connection name]:: Name your connection.
xref:dataflow-sap-hana-reference.adoc#dataflow-sap-hana-conn-connection-type[Connection type]:: Choose the SAP HANA connection type.
xref:dataflow-sap-hana-reference.adoc#dataflow-sap-hana-conn-host[Host]:: Specify the hostname or the IP address of the Sap Hana system.
xref:dataflow-sap-hana-reference.adoc#dataflow-sap-hana-conn-port[Port]:: Specify the port associated with the Sap Hana system.
xref:dataflow-sap-hana-reference.adoc#dataflow-sap-hana-conn-user[User]:: Specify the user id that will be used to connect to the Sap Hana system.
This user should have necessary privileges to access the data in the databases.
xref:dataflow-sap-hana-reference.adoc#dataflow-sap-hana-conn-password[Password]:: Specify the password for the User.
xref:dataflow-sap-hana-reference.adoc#dataflow-sap-hana-conn-database[Database]:: Collection of information that is organized so that it can be easily accessed, managed and updated.
xref:dataflow-sap-hana-reference.adoc#dataflow-sap-hana-conn-jdbc-options[JDBC options]:: Specify the options associated with the JDBC URL.
. Click *Create connection*.
