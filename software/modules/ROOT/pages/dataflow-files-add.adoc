= Add a File database connection
:last_updated: 02/19/2021
:linkattrs:
:experimental:
:page-aliases: /data-integrate/dataflow/dataflow-files-add.adoc

You can add a connection to flat files, both local and remote, using ThoughtSpot DataFlow.

Follow these steps:

include::partial$add-database-connection.adoc[]

. After you select the Files *Connection type*, the rest of the connection properties appear.
+
Depending on your choice of authentication mechanism, you may use different properties.
+
See xref:dataflow-files-reference.adoc#connection-properties[Connection properties] for details, defaults, and examples.
+

xref:dataflow-files-reference.adoc#dataflow-files-conn-connection-name[Connection name]:: Name your connection.
xref:dataflow-files-reference.adoc#dataflow-files-conn-connection-type[Connection type]:: Choose the Files connection type.
xref:dataflow-files-reference.adoc#dataflow-files-conn-file-location[File location]:: Specify the base location of the file on the server.
xref:dataflow-files-reference.adoc#dataflow-files-conn-files-on-remote-location[Files on remote location]:: Specify if the files on remote server.
xref:dataflow-files-reference.adoc#dataflow-files-conn-protocol[Protocol]:: Select the required remote server connection.
+
Mandatory field.
For remote location files only.
xref:dataflow-files-reference.adoc#dataflow-files-conn-authentication-type[Authentication type]:: Specify the authentication type for SFTP Protocol.
+
Mandatory field.
For SFTP protocol only
xref:dataflow-files-reference.adoc#dataflow-files-conn-host[Host]:: Specify the Hostname or the IP address of the remote server.
+
Mandatory field.
For remote location files only.
xref:dataflow-files-reference.adoc#dataflow-files-conn-port[Port]:: Specify the Port to connect the remote server.
+
Mandatory field.
For remote location files only.
xref:dataflow-files-reference.adoc#dataflow-files-conn-user[User]:: Specify the user to connect to remote server.
This user must have data access privileges.
For remote location files only.
xref:dataflow-files-reference.adoc#dataflow-files-conn-password[Password]:: Specify the password.
For remote location files only, when using password authentication.
xref:dataflow-files-reference.adoc#dataflow-files-conn-key-file[Key file]:: Specify the key file and its fully qualified path.
For remote location files only, when using key authentication.
xref:dataflow-files-reference.adoc#dataflow-files-conn-passphrase-for-key-file[Passphrase for key file]:: Specify the passphrase for the key file.
For remote location files only, when using key authentication.

. Click *Create connection*.
