== Keywords in other languages

Currently, we offer the following keyword languages:

* xref:keywords-da-DK.adoc[Dansk, Danish]
* xref:keywords-de-DE.adoc[Deutsch, German]
* xref:keywords-en-US.adoc[English (American)]
* xref:keywords-es-ES.adoc[Español (España), Spanish]
* xref:keywords-es-US.adoc[Español (Latinoamérica), Spanish]
* xref:keywords-fr-CA.adoc[Français (Canada), French (Canadian)]
* xref:keywords-fr-FR.adoc[Français (France), French]
* xref:keywords-it-IT.adoc[Italiano, Italian]
* xref:keywords-nl-NL.adoc[Nederland, Dutch]
* xref:keywords-nb-NO.adoc[Norsk, Norwegian]
* xref:keywords-pt-PT.adoc[Português (Portugal), Portuguese]
* xref:keywords-pt-BR.adoc[Português (Brasil), Portuguese (Brazil)]
* xref:keywords-ru-RU.adoc[русский (ограниченный выпуск), Russian (Limited Edition)]
* xref:keywords-fi-FI.adoc[Suomi, Finnish]
* xref:keywords-sv-SE.adoc[Svenska, Swedish]
* xref:keywords-zh-CN.adoc[中文 (简体), Chinese (Simplified)]
* xref:keywords-zh-HANT.adoc[中文 (繁體), Chinese (Traditional)]
* xref:keywords-ja-JP.adoc[日本語, Japanese]

To control language, date, and number formats on the ThoughtSpot UI, see how to set xref:user-profile.adoc[locale preferences in your user profile].

You can only use one set of keywords at a time. For example, if you set your locale to Deutsch, you can only use Deutsch keywords. You can't also use English keywords.
