== Keywords in Other Languages

Currently, we offer the following keyword languages:

* xref:keywords-zh-CN.adoc[Chinese (Simplified), 中文 (简体)]
* xref:keywords-da-DK.adoc[Danish, Dansk]
* xref:keywords-nl-NL.adoc[Dutch, Nederland]
* xref:keywords-fi-FI.adoc[Finnish, Suomi]
* xref:keywords-en-US.adoc[English (American)]
* xref:keywords-fr-FR.adoc[French, Français (France)]
* xref:keywords-fr-CA.adoc[French (Canadian), Français (Canada)]
* xref:keywords-de-DE.adoc[German, Deutsch]
* xref:keywords-it-IT.adoc[Italian, Italiano]
* xref:keywords-ja-JP.adoc[Japanese, 日本語]
* xref:keywords-nb-NO.adoc[Norwegian, Norsk]
* xref:keywords-pt-PT.adoc[Portuguese, Português (Portugal)]
* xref:keywords-pt-BR.adoc[Portuguese (Brazil), Português (Brasil)]
* xref:keywords-es-ES.adoc[Spanish, Español (España)]
* xref:keywords-es-US.adoc[Spanish, Español (Latinoamérica)]
* xref:keywords-sv-SE.adoc[Swedish, Svenska]

To control language, date, and number formats on the ThoughtSpot UI, see how to set xref:user-profile.adoc[locale preferences in your user profile].

You can only use one set of keywords at a time. For example, if you set your locale to Deutsch, you can only use Deutsch keywords. You can't also use English keywords.
