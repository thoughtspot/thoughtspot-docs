. Specify the sync properties for SAS:
+
See xref:dataflow-sas-reference.adoc#sync-properties[Sync properties] for details, defaults, and examples.
+
xref:dataflow-sas-reference.adoc#dataflow-sas-sync-column-delimiter[Column delimiter]:: Specify the column delimiter character.
xref:dataflow-sas-reference.adoc#dataflow-sas-sync-enclosing-character[Enclosing character]:: Specify if the text columns in the source data needs to be enclosed in quotes.
xref:dataflow-sas-reference.adoc#dataflow-sas-sync-escape-character[Escape character]:: Specify this if the text qualifier is mentioned.
This should be the character which escapes the text qualifier character in the source data.
xref:dataflow-sas-reference.adoc#dataflow-sas-sync-parallel-thread-limit[Parallel thread limit]:: Specify the number of parallel threads to unload data.
xref:dataflow-sas-reference.adoc#dataflow-sas-sync-max-ignored-rows[Max ignored rows]:: Abort the transaction after encountering 'n' ignored rows.
xref:dataflow-sas-reference.adoc#dataflow-sas-sync-ts-load-options[TS load options]:: Specifies the parameters passed with the `tsload` command, in addition to the commands already included by the application.
+
The format for these parameters is:
+
[source]
----
--<param_1_name> <optional_param_1_value>
--<param_2_name> <optional_param_2_value>
----
. Save your work by clicking *Save*.
+
Alternatively, click *Save and sync now* to save your work and sync data at the same time.
