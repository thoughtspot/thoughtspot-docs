After using ThoughtSpot DataFlow to establish a connection to an Oracle database, you can create automatic data updates, to seamlessly refresh your data.

include::partial$sync-for-databases.adoc[]

. Specify the sync properties for Oracle:
+
See xref:dataflow-oracle-reference.adoc#sync-properties[Sync properties] for details, defaults, and examples.
+
// ![Enter connection details]({{ site.baseurl }}/images/dataflow-oracle-sync.png "Enter connection details")
xref:dataflow-oracle-reference.adoc#dataflow-oracle-sync-data-extraction-mode[Data extraction mode]:: Specify the extraction type.
xref:dataflow-oracle-reference.adoc#dataflow-oracle-sync-column-delimiter[Column delimiter]:: Specify the column delimiter character.
xref:dataflow-oracle-reference.adoc#dataflow-oracle-sync-null-value[Null value]:: Specifies the string literal that should indicate the null value in the extracted data.
During the data load the column value matching this string will be loaded as null in the target.
xref:dataflow-oracle-reference.adoc#dataflow-oracle-sync-enclosing-character[Enclosing character]:: Specify if the text columns in the source data needs to be enclosed in quotes.
xref:dataflow-oracle-reference.adoc#dataflow-oracle-sync-escape-character[Escape character]:: Specify the escape character if using a text qualifier in the source data.
xref:dataflow-oracle-reference.adoc#dataflow-oracle-sync-fetch-size[Fetch size]:: Specify the number of rows to be fetched at a time and processed in memory.
If the value specified is zero then, all rows are extracted at once.
xref:dataflow-oracle-reference.adoc#dataflow-oracle-sync-max-ignored-rows[Max ignored rows]:: Abort the transaction after encountering 'n' ignored rows.
xref:dataflow-oracle-reference.adoc#dataflow-oracle-sync-ts-load-options[TS load options]:: Specify the parameters passed with the `tsload` command, in addition to the commands already included by the application.

. Save your work by selecting *Save*.
+
Alternatively, select *Save and sync now* to save your work and sync data at the same time.
