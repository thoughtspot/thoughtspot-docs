= Add a Vertica database connection
:last_updated: 09/10/2021
:linkattrs:
:experimental:
:page-aliases: /data-integrate/dataflow/dataflow-vertica-add.adoc

You can add a connection to a Vertica database using ThoughtSpot DataFlow.

Follow these steps:

include::partial$add-database-connection.adoc[]

. After you select the Vertica *Connection type*, the rest of the connection properties appear.
+
See xref:dataflow-vertica-reference.adoc#connection-properties[Connection properties] for details, defaults, and examples.
+
xref:dataflow-vertica-reference.adoc#dataflow-vertica-conn-connection-name[Connection name]:: Name your connection.

xref:dataflow-vertica-reference.adoc#dataflow-vertica-conn-connection-type[Connection type]:: Choose the Vertica connection type.

xref:dataflow-vertica-reference.adoc#dataflow-vertica-conn-host-name[Host name]:: Specify the hostname or the IP address of the Vertica system

xref:dataflow-vertica-reference.adoc#dataflow-vertica-conn-port[Port]:: Specify the port associated to the Vertica system

xref:dataflow-vertica-reference.adoc#dataflow-vertica-conn-user[User]:: Specify the user id that will be used to connect to the Vertica system. This user should have necessary privileges to access the data in the databases.

xref:dataflow-vertica-reference.adoc#dataflow-vertica-conn-password[Password]:: Specify the password for the User

xref:dataflow-vertica-reference.adoc#dataflow-vertica-conn-database[Database]:: Collection of information that is organized so that it can be easily accessed, managed, and updated.

xref:dataflow-vertica-reference.adoc#dataflow-vertica-conn-ssl-encryption[SSL encryption]:: To ensure the data is encrypted use additional security feature such as SSL.

xref:dataflow-vertica-reference.adoc#dataflow-vertica-conn-trust-store[Trust store]:: Specify the TLS/SSL client certificate store for SSL Client Authentication (2-way SSL)

xref:dataflow-vertica-reference.adoc#dataflow-vertica-conn-trust-store-password[Trust store password]:: Specify the password for the TLS/SSL client certificate.

. Click *Create connection*.
