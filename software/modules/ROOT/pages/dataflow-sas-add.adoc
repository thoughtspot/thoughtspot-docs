= Add a SAS connection
:last_updated: 10/27/2020
:experimental:
:linkattrs:
:page-aliases: /data-integrate/dataflow/dataflow-sas-add.adoc



You can add a connection to the SAS database using ThoughtSpot DataFlow.

Follow these steps:

include::partial$add-database-connection.adoc[]

. After you select the SAS *Connection type*, the rest of the connection properties appear.
+
See xref:dataflow-sas-reference.adoc#connection-properties[Connection properties] for details, defaults, and examples.
+
xref:dataflow-sas-reference.adoc#dataflow-sas-conn-connection-name[Connection name]:: Name your connection.
xref:dataflow-sas-reference.adoc#dataflow-sas-conn-connection-type[Connection type]:: Choose the SAS connection type.
xref:dataflow-sas-reference.adoc#dataflow-sas-conn-host[Host]:: Specify the hostname or the IP address of the system.
xref:dataflow-sas-reference.adoc#dataflow-sas-conn-port[Port]:: Specify the port associated with the system.
xref:dataflow-sas-reference.adoc#dataflow-sas-conn-user[User]:: Specify the user who connects to the system.
This user must have data access privileges.
xref:dataflow-sas-reference.adoc#dataflow-sas-conn-password[Password]:: Specify the password for the User.
xref:dataflow-sas-reference.adoc#dataflow-sas-conn-server-type[Server type]:: Specify the type of server to connect.
xref:dataflow-sas-reference.adoc#dataflow-sas-conn-ref-name[Ref name]:: Specify the reference name given to recognize the server.
xref:dataflow-sas-reference.adoc#dataflow-sas-conn-path[Path]:: Specify the path which is generated by server.
xref:dataflow-sas-reference.adoc#dataflow-sas-conn-version[Version]:: Specify the SAS version being connected to.

. Click *Create connection*.
