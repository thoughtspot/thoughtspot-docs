= Sync data through an SAP HANA connection
:last_updated: 9/9/2021
:experimental:
:linkattrs:
:page-aliases: /data-integrate/dataflow/dataflow-sap-hana-sync.adoc



After using ThoughtSpot DataFlow to establish a connection to an SAP HANA database, you can create automatic data updates, to seamlessly refresh your data.

include::partial$sync-for-databases.adoc[]

. Specify the sync properties for SAP HANA:
+
See xref:dataflow-sap-hana-reference.adoc#sync-properties[Sync properties] for details, defaults, and examples.
+
// ![Enter connection details]({{ site.baseurl }}/images/dataflow-sap-hana-sync.png "Enter connection details")
xref:dataflow-sap-hana-reference.adoc#dataflow-sap-hana-sync-column-delimiter[Column delimiter]:: Specify the delimiter to use as the column delimiter for the extracted data.::  Mandatory field.
xref:dataflow-sap-hana-reference.adoc#dataflow-sap-hana-sync-enclosing-character[Enclosing character]:: Specify if the text columns in the source data needs to be enclosed in quotes.
xref:dataflow-sap-hana-reference.adoc#dataflow-sap-hana-sync-escape-character[Escape character]:: Specify the escape character if using a text qualifier in the source data.
xref:dataflow-sap-hana-reference.adoc#dataflow-sap-hana-sync-fetch-size[Fetch size]:: Specify the number of rows at a time to fetch and process in memory.
If you specify zero, the system extracts all rows at once.
xref:dataflow-sap-hana-reference.adoc#dataflow-sap-hana-max-ignored-rows[Max ignored rows]:: Abort the transaction after encountering 'n' ignored rows.
xref:dataflow-sap-hana-reference.adoc#dataflow-sap-hana-sync-ts-load-options[TS load options]:: Specify the parameters passed with the `tsload` command, in addition to the commands already included by the application.

. Save your work by clicking *Save*.
+
Alternatively, click *Save and sync now* to save your work and sync data at the same time.
