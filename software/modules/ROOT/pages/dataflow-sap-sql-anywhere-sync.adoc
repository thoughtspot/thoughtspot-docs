After using ThoughtSpot DataFlow to establish a connection to an SAP SQL Anywhere database, you can create automatic data updates, to seamlessly refresh your data.

include::partial$sync-for-databases.adoc[]

. Specify the sync properties for SAP SQL Anywhere:
+
See xref:dataflow-sap-sql-anywhere-reference.adoc#sync-properties[Sync properties] for details, defaults, and examples.
+
// ![Enter connection details]({{ site.baseurl }}/images/dataflow-sap-sql-anywhere-sync.png "Enter connection details")
xref:dataflow-sap-sql-anywhere-reference.adoc#dataflow-sap-sql-anywhere-sync-column-delimiter[Column delimiter]:: Specify the column delimiter character.
xref:dataflow-sap-sql-anywhere-reference.adoc#dataflow-sap-sql-anywhere-sync-fetch-size[Fetch size]:: Specify the number of rows at a time to fetch and process in memory.
If you specify zero, the system extracts all rows at once.
xref:dataflow-sap-sql-anywhere-reference.adoc#dataflow-sap-sql-anywhere-sync-enclosing-character[Enclosing character]:: Specify if the text columns in the source data needs to be enclosed in quotes.
xref:dataflow-sap-sql-anywhere-reference.adoc#dataflow-sap-sql-anywhere-sync-escape-character[Escape character]:: Specify the escape character if using a text qualifier in the source data.
xref:dataflow-sap-sql-anywhere-reference.adoc#dataflow-sap-sql-anywhere-sync-ts-load-options[TS load options]:: Specify the parameters passed with the `tsload` command, in addition to the commands already included by the application.

. Save your work by selecting *Save*.
+
Alternatively, select *Save and sync now* to save your work and sync data at the same time.
