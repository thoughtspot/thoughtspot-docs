= Add an Azure Blob Storage connection
:last_updated: 03/09/2022
:linkattrs:
:experimental:
:page-aliases: /data-integrate/dataflow/dataflow-azure-blob-storage-add.adoc

You can add a connection to the Azure Blob Storage file system using ThoughtSpot DataFlow.

Follow these steps:

include::partial$add-database-connection.adoc[]

. After you select the *Connection type*, the rest of the connection properties appear.
+
See xref:dataflow-azure-blob-storage-reference.adoc#connection-properties[Connection properties] for details, defaults, and examples.
+
xref:dataflow-azure-blob-storage-reference.adoc#dataflow-azure-blob-storage-conn-connection-name[Connection name]::  Name your connection.
xref:dataflow-azure-blob-storage-reference.adoc#dataflow-azure-blob-storage-conn-connection-type[Connection type]:: Choose the Azure Blob Storage connection type.
xref:dataflow-azure-blob-storage-reference.adoc#dataflow-azure-blob-storage-conn-storage-type[Storage type]:: Choose the Storage type: _Azure Blob Storage_ (default) or _Data Lake GEN2 Storage_.
xref:dataflow-azure-blob-storage-reference.adoc#dataflow-azure-blob-storage-conn-account[Account]:: Azure Storage has an address that includes our unique account name which is a combination of the account name and the Azure Storage blob endpoint forms the base address for the objects in your storage account.
xref:dataflow-azure-blob-storage-reference.adoc#dataflow-azure-blob-storage-conn-authentication-type[Authentication type]:: This supports two types of authentication methods:
+
Azure account key: Choose to authenticate based on access keys and other parameters to produce an encrypted signature string that is passed on the request in the Authorization header.
+
Shared access signatures (SAS): Choose to grant fine-grained access to resources in your storage account with specified permissions and over a specified time interval.
xref:dataflow-azure-blob-storage-reference.adoc#dataflow-azure-blob-storage-conn-key[Key]:: The 512-bit access key generated while creating storage account.
xref:dataflow-azure-blob-storage-reference.adoc#dataflow-azure-blob-storage-conn-container[Container]:: Give container details that organizes a set of blobs, like a directory in a file system.
xref:dataflow-azure-blob-storage-reference.adoc#dataflow-azure-blob-storage-conn-folder[Folder]:: Give the storage folder details. Displayed only when "Storage type" is "Datalake GEN2 Storage".

. Click *Create connection*.
