Follow these steps to connect to Databricks databases:

include::partial$connections-nav1.adoc[]

. In the *Choose your data warehouse* interface, specify the fields:
+
xref:connections-databricks-reference.adoc#connection-name[Name your connection]:: For example, _MyDatabricksDatabase_
xref:connections-databricks-reference.adoc#connection-description[Describe your connections]:: The optional description of the database
Choose your data warehouse:: Select the Databricks tile
+
image::connection-databricks-add.png[Add Databricks connection]

. Click *Connect*.

. In the *Specify connection details* interface, enter the connection details for your Databricks data source.
+
image::connection-databricks-specify-details.png[]
+
xref:connections-databricks-reference.adoc#host[Host]:: Databricks server host name without the protocol.
xref:connections-databricks-reference.adoc#http-path[HTTP Path]:: Path to the host in HTTP format
xref:connections-databricks-reference.adoc#user-id[User]:: The user account name
xref:connections-databricks-reference.adoc#password[Password]:: The password of the user account
+
Refer to the xref:connections-databricks-reference.adoc[Databricks connection reference] for more information on each of the specific attributes you must enter for your connection.
