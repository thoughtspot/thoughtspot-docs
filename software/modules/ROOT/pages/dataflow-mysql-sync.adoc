= Sync data through a MySQL connection

:experimental:
:linkattrs:

include::partial$liveboards-announcement.adoc[]


After using ThoughtSpot DataFlow to establish a connection to a MySQL database, you can create automatic data updates, to seamlessly refresh your data.

include::partial$sync-for-databases.adoc[]

. Specify the sync properties for MySQL:
+
See xref:dataflow-mysql-reference.adoc#sync-properties[Sync properties] for details, defaults, and examples.
+
// ![Enter connection details]({{ site.baseurl }}/images/dataflow-mysql-sync.png "Enter connection details")
xref:dataflow-mysql-reference.adoc#dataflow-mysql-sync-column-delimiter[Column delimiter]:: Specify the column delimiter character.
xref:dataflow-mysql-reference.adoc#dataflow-mysql-sync-enclosing-character[Enclosing character]:: Specify if the text columns in the source data needs to be enclosed in quotes.
xref:dataflow-mysql-reference.adoc#dataflow-mysql-sync-escape-character[Escape character]:: Specify the escape character if using a text qualifier in the source data.
xref:dataflow-mysql-reference.adoc#dataflow-mysql-sync-fetch-size[Fetch size]:: Specify the number of rows to be fetched at a time and processed in memory.
If the value specified is zero then, all rows are extracted at once.
xref:dataflow-mysql-reference.adoc#dataflow-mysql-sync-max-ignored-rows[Max ignored rows]:: Abort the transaction after encountering 'n' ignored rows.
xref:dataflow-mysql-reference.adoc#dataflow-mysql-sync-ts-load-options[TS load options]:: Specify the parameters passed with the `tsload` command, in addition to the commands already included by the application.

. Save your work by clicking *Save*.
+
Alternatively, click *Save and sync now* to save your work and sync data at the same time.
