= Sync data through Google Cloud Storage connection
:last_updated: 7/7/2020
:experimental:
:linkattrs:
:page-aliases: /data-integrate/dataflow/dataflow-google-cloud-storage-sync.adoc



After using ThoughtSpot DataFlow to establish a connection to a Google Cloud Storage file system, you can create automatic data updates, to seamlessly refresh your data.

include::partial$sync-for-file-systems.adoc[]

. Specify the sync properties for files:
+
See xref:dataflow-google-cloud-storage-reference.adoc#sync-properties[Sync properties] for details, defaults, and examples.
+
xref:dataflow-google-cloud-storage-reference.adoc#dataflow-google-cloud-storage-sync-file-name[File name]:: Specify name of the file.
xref:dataflow-google-cloud-storage-reference.adoc#dataflow-google-cloud-storage-sync-enable-archive-on-success[Enable archive on success]:: Specify if file needs to be archived once it is succeeded.
+
Optional field.
xref:dataflow-google-cloud-storage-reference.adoc#dataflow-google-cloud-storage-sync-delete-file-on-success[Delete file on success]:: Specify if file needs to be deleted after execution is successful.
+
Optional field.
xref:dataflow-google-cloud-storage-reference.adoc#dataflow-google-cloud-storage-sync-column-delimiter[Column delimiter]:: Specify the column delimiter character.
xref:dataflow-google-cloud-storage-reference.adoc#dataflow-google-cloud-storage-sync-skip-header-rows[Skip header rows]:: Skip the number of header rows specified while loading the data.
+
Optional field.
xref:dataflow-google-cloud-storage-reference.adoc#dataflow-google-cloud-storage-sync-compression[Compression]:: Specify this if the file is compressed and what kind of compressed file it is.
+
Mandatory field.
xref:dataflow-google-cloud-storage-reference.adoc#dataflow-google-cloud-storage-sync-row-delimiter[Row delimiter]:: Specifies the character to be used to indicate the end of the row in the extracted data.
xref:dataflow-google-cloud-storage-reference.adoc#dataflow-google-cloud-storage-sync-enclosing-character[Enclosing character]:: Specify if the text columns in the source data is enclosed in quotes.
xref:dataflow-google-cloud-storage-reference.adoc#dataflow-google-cloud-storage-sync-escape-character[Escape character]:: Specify the escape character if using a text qualifier in the source data.
xref:dataflow-google-cloud-storage-reference.adoc#dataflow-google-cloud-storage-sync-null-value[Null value]:: Specifies the string literal indicates the null value for a column.
During the data load, the column value matching this string will be loaded as null in the target.
xref:dataflow-google-cloud-storage-reference.adoc#dataflow-google-cloud-storage-sync-date-style[Date style]:: Specifies how to interpret the date format.
+
Optional field.
xref:dataflow-google-cloud-storage-reference.adoc#dataflow-google-cloud-storage-sync-date-delimiter[Date delimiter]:: Specifies the separator used in the date format.
xref:dataflow-google-cloud-storage-reference.adoc#dataflow-google-cloud-storage-sync-time-style[Time style]:: Specifies the format of the time portion in the data.
xref:dataflow-google-cloud-storage-reference.adoc#dataflow-google-cloud-storage-sync-time-delimiter[Time delimiter]:: Specifies the character used as separate the time components.
xref:dataflow-google-cloud-storage-reference.adoc#dataflow-google-cloud-storage-sync-decimal-delimiter[Decimal delimiter]:: Specify the decimal delimiter for float/double/numeric data types.
xref:dataflow-google-cloud-storage-reference.adoc#dataflow-google-cloud-storage-sync-skip-trailer-rows[Skip trailer rows]:: Skip the number of trailer rows specified while loading the data.
+
Optional field.
xref:dataflow-google-cloud-storage-reference.adoc#dataflow-google-cloud-storage-sync-ts-load-options[TS load options]:: Specify the parameters passed with the `tsload` command, in addition to the commands already included by the application.
xref:dataflow-google-cloud-storage-reference.adoc#dataflow-google-cloud-storage-sync-boolean-representation[Boolean representation]:: Specifies the representation of data in the boolean field.
+
Optional field.

. Save your work by clicking *Save*.
+
Alternatively, click *Save and sync now* to save your work and sync data at the same time.
