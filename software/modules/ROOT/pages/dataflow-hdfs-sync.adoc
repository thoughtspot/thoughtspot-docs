= Sync data through HDFS connection
:last_updated: 7/7/2020
:experimental:
:linkattrs:
:page-aliases: /data-integrate/dataflow/dataflow-hdfs-sync.adoc



After using ThoughtSpot DataFlow to establish a connection to an lahdfs file system, you can create automatic data updates, to seamlessly refresh your data.

include::partial$sync-for-file-systems.adoc[]

. Specify the sync properties for files:
+
See xref:dataflow-hdfs-reference.adoc#sync-properties[Sync properties] for details, defaults, and examples.
+
xref:dataflow-hdfs-reference.adoc#dataflow-hdfs-sync-column-delimiter[Column delimiter]:: Specify the column delimiter character.
xref:dataflow-hdfs-reference.adoc#dataflow-hdfs-sync-enable-archive-on-success[Enable archive on success]:: Specify if data needs to be archived once it is succeeded.::  Optional field.
xref:dataflow-hdfs-reference.adoc#dataflow-hdfs-sync-delete-on-success[Delete on success]:: Specify if data needs to be deleted after execution is successful.
+
Optional field.
xref:dataflow-hdfs-reference.adoc#dataflow-hdfs-sync-compression[Compression]:: Specify this if the file is compressed and what kind of compressed file it is.
+
Mandatory field.
xref:dataflow-hdfs-reference.adoc#dataflow-hdfs-sync-enclosing-character[Enclosing character]:: Specify if the text columns in the source data needs to be enclosed in quotes.
xref:dataflow-hdfs-reference.adoc#dataflow-hdfs-sync-escape-character[Escape character]:: Specify the escape character if using a text qualifier in the source data.
xref:dataflow-hdfs-reference.adoc#dataflow-hdfs-sync-null-value[Null value]:: Specify the string literal that represents NULL values in data.
During the data load, the column value that matches this string loads as NULL into ThoughtSpot.
xref:dataflow-hdfs-reference.adoc#dataflow-hdfs-sync-date-style[Date style]::
Specifies how to interpret the date format.
+
Optional field.
xref:dataflow-hdfs-reference.adoc#dataflow-hdfs-sync-date-delimiter[Date delimiter]:: Specifies the separator used in the date format (only default delimiter is supported).
xref:dataflow-hdfs-reference.adoc#dataflow-hdfs-sync-time-style[Time style]:: Specifies the format of the time portion in the data.
xref:dataflow-hdfs-reference.adoc#dataflow-hdfs-sync-time-delimiter[Time delimiter]:: Specifies the character used as separate the time components.
(Only default delimiter is supported)
+
Optional field.
xref:dataflow-hdfs-reference.adoc#dataflow-hdfs-sync-ts-load-options[TS load options]:: Specify the parameters passed with the `tsload` command, in addition to the commands already included by the application.

. Save your work by clicking *Save*.
+
Alternatively, click *Save and sync now* to save your work and sync data at the same time.
