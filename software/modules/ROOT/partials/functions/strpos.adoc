[#strpos]
`strpos`::
Returns the numeric position of the first occurrence of the second string in the first string. If using an external cloud data warehouse xref:connections.adoc[connection], the position starts at 1, and 0 indicates not found. If using ThoughtSpot's internal in-memory database (Falcon), the position starts at 0, and -1 indicates not found.
Examples;;
+
----
In-memory database: strpos ('haystack_with_needles', 'needle') = 14
External database: strpos ('haystack_with_needles', 'needle') = 15
strpos (complaint, 'lawyer')
----
