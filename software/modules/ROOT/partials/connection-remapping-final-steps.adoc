. In the *Remapping* interface, click *image:sign-download.png[] Download*.
+
This saves the definition of your connection to your local directory.

. Edit the file as needed, and save it.
+
When changing a column's data type, refer to xref:data-type-remap.adoc[] to determine the correct type and format to represent the data type in the external databases. Ensure that the changed data type matches the one in the external source, and that ThoughtSpot supports it.

. Finally, click *Browse* your files, and upload your edited mapping file to update your connection.
