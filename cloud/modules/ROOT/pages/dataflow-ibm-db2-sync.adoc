= Sync data through an IBM Db2 connection
:last_updated: 9/10/2021
:experimental:
:linkattrs:
:page-aliases: /data-integrate/dataflow/dataflow-ibm-db2-sync.adoc



After using ThoughtSpot DataFlow to establish a connection to an IBM Db2 database, you can create automatic data updates, to seamlessly refresh your data.

include::partial$sync-for-databases.adoc[]

. Specify the sync properties for IBM Db2:
+
See xref:dataflow-ibm-db2-reference.adoc#sync-properties[Sync properties] for details, defaults, and examples.
+
// ![Enter connection details]({{ site.baseurl }}/images/dataflow-ibm-db2-sync.png "Enter connection details")
xref:dataflow-ibm-db2-reference.adoc#dataflow-ibm-db2-sync-data-extraction-mode[Data extraction mode]:: Specify the extraction type.
xref:dataflow-ibm-db2-reference.adoc#dataflow-ibm-db2-sync-column-delimiter[Column delimiter]:: Specify the column delimiter character.
xref:dataflow-ibm-db2-reference.adoc#dataflow-ibm-db2-sync-null-value[Null value]:: Specifies the string literal that should indicate the null value in the extracted data.
During the data load the column value matching this string will be loaded as null in the target.
xref:dataflow-ibm-db2-reference.adoc#dataflow-ibm-db2-sync-enclosing-character[Enclosing character]:: Specify if the text columns in the source data needs to be enclosed in quotes.
xref:dataflow-ibm-db2-reference.adoc#dataflow-ibm-db2-sync-escape-character[Escape character]:: Specify the escape character if using a text qualifier in the source data.
xref:dataflow-ibm-db2-reference.adoc#dataflow-ibm-db2-sync-fetch-size[Fetch size]:: Specify the number of rows to be fetched at a time and processed in memory.
If the value specified is zero then, all rows are extracted at once.
xref:dataflow-ibm-db2-reference.adoc#dataflow-ibm-db2-sync-max-ignored-rows[Max ignored rows]:: Abort the transaction after encountering 'n' ignored rows.
xref:dataflow-ibm-db2-reference.adoc#dataflow-ibm-db2-sync-ts-load-options[TS load options]:: Specify the parameters passed with the `tsload` command, in addition to the commands already included by the application.

. Save your work by clicking *Save*.
+
Alternatively, click *Save and sync now* to save your work and sync data at the same time.
