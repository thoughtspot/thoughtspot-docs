= Add a Greenplum database connection
:last_updated: 3/22/2021
:page-aliases: /data-integrate/dataflow/dataflow-greenplum-add.adoc
:linkattrs:
:experimental:

You can add a connection to a Greenplum database using ThoughtSpot DataFlow.

Follow these steps:

include::partial$add-database-connection.adoc[]

. After you select the Greenplum *Connection type*, the rest of the connection properties appear.
xref:dataflow-greenplum-reference.adoc#dataflow-greenplum-conn-connection-name[Connection name]::
Name your connection.
xref:dataflow-greenplum-reference.adoc#dataflow-greenplum-conn-connection-type[Connection type]::
Choose the Greenplum connection type.
xref:dataflow-greenplum-reference.adoc#dataflow-greenplum-conn-host[Host]::
Specify the hostname or the IP address of the Greenplum system
xref:dataflow-greenplum-reference.adoc#dataflow-greenplum-conn-port[Port]::
Specify the port associated to the Greenplum system
xref:dataflow-greenplum-reference.adoc#dataflow-greenplum-conn-user[User]::
Specify the user id that will be used to connect to the Greenplum system.
This user should have necessary privileges to access the data in the databases.
xref:dataflow-greenplum-reference.adoc#dataflow-greenplum-conn-password[Password]::
Specify the password for the User
xref:dataflow-greenplum-reference.adoc#dataflow-greenplum-conn-database[Database]::
Collection of information that is organized so that it can be easily accessed, managed and updated.
xref:dataflow-greenplum-reference.adoc#dataflow-greenplum-conn-version[Version]::
Specify the Greenplum version being connected to
xref:dataflow-greenplum-reference.adoc#dataflow-greenplum-conn-jdbc-options[JDBC options]::
Specify the options associated with the JDBC URL.
+
See xref:dataflow-greenplum-reference.adoc#connection-properties[Connection properties] for details, defaults, and examples.
. Click *Create connection*.
