= Add an Apache Parquet connection
:last_updated: 02/19/2021
:linkattrs:
:experimental:
:page-aliases: /data-integrate/dataflow/dataflow-apache-parquet-add.adoc

You can add a connection to the Apache Parquet file system using ThoughtSpot DataFlow.

Follow these steps:

include::partial$add-database-connection.adoc[]

. After you select the *Connection type*, the rest of the connection properties appear.
+
See xref:dataflow-apache-parquet-reference.adoc#connection-properties[Connection properties] for details, defaults, and examples.
+
xref:dataflow-apache-parquet-reference.adoc#dataflow-apache-parquet-conn-connection-name[Connection name]::  Name your connection.
xref:dataflow-apache-parquet-reference.adoc#dataflow-apache-parquet-conn-connection-type[Connection type]::  Choose the Apache Parquet connection type.
xref:dataflow-apache-parquet-reference.adoc#dataflow-apache-parquet-conn-file-location[File location]::  Specify the base location of the file on the server.
xref:dataflow-apache-parquet-reference.adoc#dataflow-apache-parquet-conn-files-on-remote-location[Files on remote location]::  Specify if the files are on a remote server.
xref:dataflow-apache-parquet-reference.adoc#dataflow-apache-parquet-conn-protocol[Protocol]::  Select the required remote server connection.
+
For remote location files only.
xref:dataflow-apache-parquet-reference.adoc#dataflow-apache-parquet-conn-authentication-type[Authentication type]:: Specify the authentication type for SFTP Protocol.
+
For SFTP protocol only.
xref:dataflow-apache-parquet-reference.adoc#dataflow-apache-parquet-conn-host[Host]::  Specify the Hostname or the IP address of the remote server.
+
For remote location files only.
xref:dataflow-apache-parquet-reference.adoc#dataflow-apache-parquet-conn-port[Port]::  Specify the Port to connect the remote server
+
For remote location files only.
xref:dataflow-apache-parquet-reference.adoc#dataflow-apache-parquet-conn-user[User]::  Specify the user to connect to remote server.
This user must have data access privileges.
+
For remote location files only.
xref:dataflow-apache-parquet-reference.adoc#dataflow-apache-parquet-conn-password[Password]::  Specify the password.
+
For remote location files only, when using password authentication.
xref:dataflow-apache-parquet-reference.adoc#dataflow-apache-parquet-conn-key-file[Key file]::  Specify the key file and its fully qualified path.
+
For remote location files only, when using key authentication.
xref:dataflow-apache-parquet-reference.adoc#dataflow-apache-parquet-conn-passphrase-for-key[Passphrase for key]::  Specify the passphrase for the key file.
+
For remote location files only, when using key authentication.

. Click *Create connection*.
