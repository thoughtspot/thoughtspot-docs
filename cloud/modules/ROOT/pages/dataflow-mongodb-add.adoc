= Add a MongoDB database connection
:last_updated: 9/10/2021
:experimental:
:linkattrs:
:page-aliases: /data-integrate/dataflow/dataflow-mongodb-add.adoc



You can add a connection to a MongoDB database using ThoughtSpot DataFlow.

Follow these steps:

include::partial$add-database-connection.adoc[]

. After you select the MongoDB *Connection type*, the rest of the connection properties appear.
+
See xref:dataflow-mongodb-reference.adoc#connection-properties[Connection properties] for details, defaults, and examples.
+
xref:dataflow-mongodb-reference.adoc#dataflow-mongodb-conn-connection-name[Connection name]:: Name your connection.
xref:dataflow-mongodb-reference.adoc#dataflow-mongodb-conn-connection-type[Connection type]:: Choose the MongoDB connection type.
xref:dataflow-mongodb-reference.adoc#dataflow-mongodb-conn-host-[Host]:: Specify the hostname or the IP address of the MongoDB system.
xref:dataflow-mongodb-reference.adoc#dataflow-mongodb-conn-port-[Port]:: Specify the port associated to the MongoDB system.
xref:dataflow-mongodb-reference.adoc#dataflow-mongodb-conn-user-[User]:: Specify the user id that will be used to connect to the MongoDB system.
This user should have necessary privileges to access the data in the databases.
xref:dataflow-mongodb-reference.adoc#dataflow-mongodb-conn-password-[Password]:: Specify the password for the User.
xref:dataflow-mongodb-reference.adoc#dataflow-mongodb-conn-database-[Database]:: Collection of information that is organized so that it can be easily accessed, managed and updated.
+
////
xref:dataflow-mongodb-reference.adoc#dataflow-mongodb-sync-flatten-arrays[Flatten arrays]:: The FlattenArrays property can be used to flatten the elements of nested arrays into columns of their own.
Set FlattenArrays to the number of elements you want to return from nested arrays.
////
xref:dataflow-mongodb-reference.adoc#dataflow-mongodb-sync-row-scan-depth[Row scan depth]:: The maximum number of rows to scan to look for the columns available in a table.
Set this property to gain more control over how the provider applies data types to collections.
xref:dataflow-mongodb-reference.adoc#dataflow-mongodb-conn-jdbc-options[JDBC options]:: Specify the options associated with the JDBC URL.

. Click *Create connection*.
